# `vim commands`

#### navigation

`h`,`j`,`k`,`l` - left, down, right, up  
`H`,`M`,`L` - go to top, middle, bottom of page  

#### editing

`i`/`a` - insert mode  
`A` - insert mode at end of line  
`d` - delete
- `w` - delete to end of word  (**inc.** current char)
- `e` - delete to end of word  (**exc.** current char)
- `$` - delete to end of line
